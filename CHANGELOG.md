## [1.0.0] — 2025-02-28

### Added
- **Smart Activity Tracking** — Event-driven tab monitoring with intelligent idle detection
- **Focus Mode** — Strict site blocking with task-based unlock and cognitive friction barriers
- **Domain Control** — Temporary blocks, daily limits, and scheduled restrictions
- **Behavioral Insights** — Distraction loop detection, peak productivity window identification
- **Analytics Engine** — Weekly trends, category breakdowns, hourly activity heatmaps
- **Productivity Score** — Composite scoring from focus time, session consistency, and patterns
- **Dashboard** — Full-featured analytics dashboard with charts and trend visualization
- **Onboarding** — Guided setup flow for new users
- **Achievement System** — Gamified milestones for productivity streaks
- **Landing Page** — Premium SaaS-quality marketing site with 3D effects, typewriter hero, animated counters
- **Author Section** — Personal branding with social links
- **FAQ Accordion** — Common questions with smooth expand/collapse animations
- **Floating CTA** — Scroll-triggered sticky call-to-action pill
- **Page Transitions** — Smooth framer-motion transitions between routes
- **Dark/Light Mode** — Full theme support with semantic design tokens
- **Mobile Responsive** — Optimized for all viewport sizes
- **Privacy-First Architecture** — Zero data collection, 100% local storage
- **Manifest V3** — Modern Chrome extension architecture
